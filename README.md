# Eco Vert Guadeloupe

Site vitrine de **Eco Vert Guadeloupe** — entreprise d'espaces verts et de jardinage en Guadeloupe.
Pour une Guadeloupe plus verte. 📞 0690 98 75 19

## Lancer le site

Ouvrez simplement `index.html` dans un navigateur, ou servez le dossier :

```bash
python3 -m http.server 8000
# puis ouvrez http://localhost:8000
```

## Structure

- `index.html` — page principale (accueil, services, à propos, zone, contact)
- `styles.css` — styles responsive (mobile-first + ordinateur)
- `script.js` — menu mobile et année automatique
- `assets/logo.svg` — logo (provisoire)

## Remplacer le logo

Le fichier `assets/logo.svg` est un logo provisoire. Pour mettre votre vrai logo :
1. Déposez votre image dans `assets/` (ex. `logo.png`)
2. Remplacez `assets/logo.svg` par `assets/logo.png` dans `index.html`
