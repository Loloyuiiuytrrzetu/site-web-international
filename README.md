# ECO VERT GUADELOUPE — Site web

Site vitrine pour **ECO VERT GUADELOUPE** : terrassement, nettoyage de terrain, mur en enrochement, évacuation, aménagement, travaux de pelle, espace vert, jardinier, paysagiste et élagage en Guadeloupe.

## Contenu
- `index.html` — page principale (responsive mobile, optimisée SEO)
- `styles.css` — styles et version mobile
- `script.js` — menu mobile + petites interactions
- `images/` — illustrations SVG (pelle, terrassement, espace vert, logo)
- `robots.txt` / `sitemap.xml` — référencement Google

## Mettre le site en ligne (gratuit)
1. Créer un dépôt GitHub public et y pousser ces fichiers.
2. Activer **GitHub Pages** : Settings → Pages → Branch `main` → `/root`.
3. (Recommandé) Inscrire le site sur **Google Search Console** et soumettre `sitemap.xml`.
4. Créer une fiche **Google Business Profile** pour apparaître en premier dans les recherches locales en Guadeloupe.

## Personnalisation
- Remplacez les illustrations SVG du dossier `images/` par vos vraies photos de chantier (`.png` ou `.jpg`) pour plus d'impact et de confiance.
- Mettez à jour l'adresse du site (`ecovertguadeloupe.fr`) dans `index.html`, `robots.txt` et `sitemap.xml` une fois votre nom de domaine choisi.
